Secure Digital Identity Wallet Frontend

Design a modern, professional, fully responsive web application UI for a system called:

AegisDrive – Secure Digital Identity Wallet

The system is a government-grade digital identity and behavioural compliance monitoring platform for driving licences and identity documents.

The platform integrates:

Digital Identity Wallet
Driving Licence Management
Behavioural Risk Prediction (AI)
Secure Passport Verification

The frontend will be implemented using React (JavaScript) and communicates with a FastAPI backend API.

The interface must support Role-Based Access Control (RBAC) with three user roles:

Driver
Police Officer
Administrator

The system must support light mode and dark mode with a toggle in the navigation bar.

Design the interface using a clean, modern dashboard style suitable for a secure government analytics system.

The system must be fully responsive for desktop, tablet, and mobile devices.

Global Application Layout

Authenticated pages should include:

Top Navigation Bar
Left Sidebar Navigation
Main Content Area

Top Navbar Components:

System Logo
System Name: AegisDrive
Dark / Light Mode Toggle
Notification Icon
User Profile Dropdown

Sidebar navigation should change depending on the user role.

User Roles and Access Control
Driver

Drivers can only access their own identity credentials and behavioural data.

Drivers cannot search other drivers or access system databases.

Drivers can view:

Driver Dashboard
Identity Wallet
Digital Behaviour Passport
Behaviour Analysis

Police Officer

Police officers use the system for identity and licence verification during inspections.

Police cannot access the full database.

Police can:

Scan QR codes
Verify digital identity credentials
View compliance and risk status

Police cannot:

Search driver records
Access admin tools

Administrator

Administrators have full system access.

Admins can:

Search drivers
View identity documents
Monitor AI predictions
View system logs
Manage users
