# AegisDrive - Secure Digital Identity Wallet

A modern, professional government-grade digital identity and behavioural compliance monitoring platform for driving licences.

## Features

### Authentication
- Role-based login system (Driver, Police Officer, Administrator)
- Secure government identity system

### User Roles

#### Driver
- Personal behaviour monitoring dashboard
- AI risk assessment with suspension probability gauge
- Behaviour metrics and trends
- Digital behaviour passport (Apple Wallet-style)
- QR code verification
- Compliance status tracking

#### Police Officer
- Roadside verification console
- QR code scanner interface
- Manual passport token verification
- Real-time driver information display
- Verification status (VALID, EXPIRED, TAMPERED)

#### Administrator
- Full system monitoring dashboard
- Driver search and management
- AI behaviour analysis with explainable AI
- System-wide analytics and trends
- Activity logs and audit trails
- User management

### Key Components

- **Analytics Cards**: Reusable metric display components
- **Risk Gauge**: Visual circular gauge for risk assessment
- **Charts**: Line charts, pie charts, and bar charts for data visualization
- **Dark Mode**: Full dark/light theme support with toggle
- **Responsive Design**: Mobile, tablet, and desktop optimized layouts

## Quick Start

1. Login with any username/password
2. Select your role (Driver, Police, or Admin)
3. Explore the dashboard and features
